二、ObjectC环境配置

手动配置.framework形式开发包

SDK中包含了 HoriLXJSDK.framework、WebRTC.framework、HoriLXJSDK.bundle三个文件。

1.在General -> Embedded Binaries添加WebRTC.framework：
2.在Build Phases -> Line Binary and Libraries添加：

                HoriLXJSDK.framework
                libsqlite3.tbd
                libstdc++.tbd
                libz.tbd
3.Build Phases标签添加HoriLXJSDL.bundle

                4.Build Settings标签
                Enable Bitcode选NO
                Other Linker Flags添加：-ObjC

            

                5.在项目plist文件中添加相机和麦克风的访问权限：
                访问相机：Privacy - Camera Usage Description
                访问麦克风：Privacy - Microphone Usage Description
                进入后台模式：Required background modes - App downloads content in response to push notifications
